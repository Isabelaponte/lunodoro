Lunodoro Rotas

O Lunodoro é um sistema de organização de tarefas que permite que os usuários se cadastrem, criem e gerenciem listas de tarefas de forma eficiente. Cada usuário pode criar várias listas, que armazenam múltiplas tarefas e permitem o acompanhamento do tempo focado usando o método Pomodoro.

As informações a serem enviadas no corpo da requisição podem ser do tipo application/json.

Usuários

URL de requisição: https://lunodoro/usuarios

Método Endpoint Descrição Resultado
GET GET /usuarios/{id} Obtém os dados do usuário logado, requer id 
{
    "id": 1,
    "nome_usuario": "Guilherme",
    "email": "Gui@example.com",
    "dt_criacao_conta": "2024-10-20 22:49:43"
}
POST POST /usuarios Cadastra um novo usuário, requer nome, email e senha 
{
    "msg": "Usuario criado com sucesso!"
}
POST POST /usuarios/login Realiza o login de um usuário e retorna seu identificador, requer email e senha 
{
    "id": "1"
}

Tipos de lista

URL de requisição: https://lunodoro/tipoLista

Método Endpoint Descrição Resultado
GET GET /tipoLista Obtém todas os tipos de lista disponíveis da apliacação 
[
    {
        "id": 1,
        "descricao": "pessoal"
    }
    ...
]
GET GET /tipoLista/{descricao} Obtém os dados de um tipo de lista, requer descricao 
{
"id": 3,
"descricao": "estudo"
}

Lista

URL de requisição: http://localhost/Lunodoro/usuarios/{id_usuario}/lista

Método Endpoint Descrição Resultado
GET GET usuarios/{id_usuario}/lista Obtém os dados de uma lista, requer id_usuario como parâmetro na requisição 
    {
        "msg": "Listas encontradas com sucesso!",
        "data": [
            {
                "id": 2,
                "id_usuario": 2,
                "nome_lista": "Microsserviço com GO e Kafka",
                "descricao": "Construindo microsserviços com Golang e Kafka",
                "id_tipo_lista": 2,
                "dt_criacao": "2024-10-20 22:58:28",
                "dt_atualizacao": "2024-10-20 23:24:41"
            }
        ]
    }
POST POST usuarios/{id_usuario}/lista Cria uma nova lista, requer id_usuario, nome_lista, descrição, id_tipo_lista 
    {
        "msg": "Lista criada com sucesso!"
    }
PUT PUT usuarios/{id_usuario}/lista/{id} Altera os dados de uma lista, requer id_usuario, id, nome_lista, descrição, id_tipo_lista 
    {
        "msg": "Lista atualizada com sucesso!",
        "data": {
            "id_usuario": "2",
            "id": "2",
            "nome_lista": "Microsserviço com GO e Kafka",
            "descricao": "Construindo microsserviços com Golang e Kafka",
            "id_tipo_lista": "2"
        }
    }
DELETE DELETE usuarios/{id_usuario}/lista/{id} Remove uma lista, requer id_usuario, id como parâmetros na requisição 
    {
        "msg": "Lista removida com sucesso!",
        "data": {
            "id_usuario": "1",
            "id": "5"
        }
    }

Tarefa

URL de requisição: http://localhost/Lunodoro/usuarios/{id_usuario}/lista/{id_lista}/tarefa

Método Endpoint Descrição Resultado
GET GET usuarios/{id_usuario}/lista/{id_lista}/tarefa/{id_tarefa} Obtém os dados de uma tarefa específica, requer id_usuario, id_lista e id_tarefa como parâmetros na requisição 
{
    "msg": "Tarefa encontrada com sucesso!",
    "data": {
        "id": 1,
        "nome": "fazendo testes",
        "descricao": "testando API com Postman",
        "dt_inicio": "2024-10-21 19:45:40",
        "dt_final": "2024-11-02 12:00:00",
        "duracao": null,
        "status": "em processo"
    }
}
POST POST usuarios/{id_usuario}/lista/{id_lista}/tarefa Cria uma nova tarefa, requer nome, descrição, dt_final, status, e id_lista 
{
    "msg": "Tarefa criada com sucesso!",
    "data": {
        "nome": "fazendo testes",
        "descricao": "testando API com Postman",
        "dt_final": "2024-11-02 12:00:00",
        "status": "em processo"
    }
}
PUT PUT usuarios/{id_usuario}/lista/{id} Atualiza os dados de uma tarefa, requer id_usuario, id_lista, id_tarefa, nome, descrição, dt_final, e status 
{
    "msg": "Tarefa atualizada com sucesso!",
    "data": {
        "id": 1,
        "nome": "Fazer validação de rotas PHP",
        "descricao": "Testando API com Postman",
        "dt_final": "2024-11-02 12:00:00",
        "status": "concluida"
    }
}
DELETE DELETE usuarios/{id_usuario}/lista/{id} Remove uma tarefa, requer id_usuario, id_lista e id_tarefa como parâmetros 
 {
    "msg": "Tarefa removida com sucesso!",
    "data": {
        "id": 3
    }
}
