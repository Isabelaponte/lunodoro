Lunodoro Rotas

O Lunodoro é um sistema de organização de tarefas que permite que os usuários se cadastrem, criem e gerenciem listas de tarefas de forma eficiente. Cada usuário pode criar várias listas, que armazenam múltiplas tarefas e permitem o acompanhamento do tempo focado usando o método Pomodoro.

As informações a serem enviadas no corpo da requisição podem ser do tipo application/json.

Usuários

URL de requisição: https://lunodoro/usuarios/

Método Endpoint Descrição Exemplo
POST POST /usuarios Cadastra um novo usuário, requer nome, email e senha 
{
    "nome": "isabela",
    "email": "isabela@example.com",
    "senha": "senhaSegura"
}
GET GET /meusDados Obtém os dados do usuário logado, requer email e senha 
{
    "id": 1,
    "email": "ryan@example.com",
    "senha": "senhaSegura",
    "dt_criacao": "2024-10-08 14:36:19"
}
POST POST /login Realiza o login de um usuário, requer email e senha 
{
    "email": "guilherme@example.com",
    "senha": "senhaSegura"
}
